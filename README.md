# ML \(Machine Learning\)

## 임베디드 개발자의 TensorFlow 학습하기

### [목차 바로가기](SUMMARY.md)

### 학습 목표

* 평범한 Low-Level Language 임베디드 개발자의 python과 TensorFlow의 학습 기록
* TensorFlow, python을 전혀 모르는 시각으로 출발하는 정리의 시간
